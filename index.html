퍼블리셔 코멘트

  • 로고는 비슷한 폰트로 제가 넣어놨습니다. 폰트 라이센스는 확인하지 않았기 때문에 그대로 사용하실 경우 라이센스 검토가 필요합니다.
  • 폰트 : Gemunu Libre / 라이센스 : https://openfontlicense.org/
  • 메인페이지의 상품 카테고리 슬라이드 모달 뎁스 데모는 메인페이지 하단의 객체 데이터와 common.js의 setCateModalList함수의 동작 방식을 확인해주세요.
  • 슬라이드 모달 생성시 객체 데이터 name 값에 따라 랜더링 요소가 구분되어 있습니다. 추가하거나 삭제하실 때 해당 if 구문 수정해 주시면 됩니다.
  • select는 슬라이드 모달로 기능 치환을 했습니다. open-select 클래스를 가진 엘리먼트 구조를 맞춰 주시면 됩니다. 첫번째 value가 ""인 값은 해당 슬라이드 모달의 제목이 되는 부분이니 생략하지 않으셔야 합니다.
  • 모든 스크립트는 사용하셔도 되지만 데모 구현 요소로 스크립트 동작 방식에 대한 수정은 지원해 드리지 않습니다.
  • 기타 문의는 언제든지 편하게 연락 부탁드립니다.
